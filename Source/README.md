# Web Based GUI

## Important Info

### Used Languages:
HTML, CSS, Javascript, Python

### Electron Version:
electron@v22.1.0

### Required Stuff:
* need node.js for electron
* run `npm start` in the correct directory to start gui

### NPM Installs:
* npm install electron@v22.1.0
* npm install socket.io-client
* npm install screenshot-desktop
* npm install uuid
